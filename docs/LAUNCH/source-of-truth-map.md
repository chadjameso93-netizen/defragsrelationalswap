# Source of Truth Map

In cases where architectures or design specifications conflict, DEFRAG defers to the following hierarchy:

1. **Behavioral Truth:** Canonical Internal Documentation + AI Reasoning Pipeline.
   - *Defrag is a relational reasoning system.* Birth-based data is a weaker prior than events/patterns/user feedback. Output must stay calm, grounded, non-diagnostic, uncertainty-aware, and human. 
2. **Launch Architecture:** Production SaaS Build and Release Document Set.
   - *Next.js web layer + Supabase + Stripe + OpenAI + MCP layer.* No massive infra drift.
3. **Brand / Public UX Direction:** Public Site and Release Pack + approved homepage copy direction.
   - *Dark, structured, minimal.* Pricing is structured as intelligence infrastructure, not rounded SaaS apps. Terminology strictly utilizes **Dynamics** and **Insights**.
4. **Repo Assessment:** Current repo state and branch cleanliness as provided by the operator.
